# GameGenie‑X Code Alphabet

GameGenie‑X uses a custom 32‑symbol alphabet designed for clarity, compactness, and retro flavor.

## Alphabet

A B C D E F G H J K L M N P R S T V W X Y Z 2 3 4 5 6 7 8 9


## Rationale

- Avoids ambiguous characters (`0`, `1`, `O`, `I`)
- 32 symbols → each symbol encodes **5 bits**
- Compact codes with a nostalgic aesthetic
- Easy to read, type, and share

## Symbol Mapping

Each symbol maps to a 5‑bit value (0–31).  
The mapping table is deterministic and defined in the core encoder.
